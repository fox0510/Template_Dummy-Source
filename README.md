# Template_Dummy [Source] MAIN

Source files for the Steam Workshop addon:

[Template_Dummy [PM]](https://steamcommunity.com/sharedfiles/filedetails/?id=3738014043)

## Download

Go to the [Releases](https://github.com/fox0510/Template_Dummy-Source/releases) page for the latest archive.

## Includes

- Playermodel source files
- C_Arms source files
- QC files
- Materials
- Blender-exported source files (.gltf)
- Compiled models
- Example addon files

## Purpose

Template_Dummy is a free playermodel template and learning resource for Garry's Mod creators.

Feel free to use it as a base for your own projects.
